# PandaWiki 上线配置清单（部署后逐项填）

## A. 登录
- [ ] 控制台可访问：`http://<主机IP>:2443`
- [ ] 用 `admin` + 安装时打印的随机密码登录
- [ ] 首次登录后**修改 admin 密码**

## B. 对话模型（← 接入你们的内部 API key）
- [ ] 模型类型：OpenAI 兼容 / DeepSeek / 自定义（选与你们网关匹配的）
- [ ] **Base URL**：`https://<内网模型网关>/v1`（确认是 OpenAI 兼容路径）
- [ ] **API Key**：内部 key
- [ ] **模型名**：网关上实际可用的对话模型
- [ ] 点「测试连通」/发一条测试问答，确认能正常返回
- [ ] 确认该请求**只走内网**（抓包或看网关日志验证不出网）

## C. 向量 / 重排模型
- [ ] 向量模型 `bge-m3`：内置容器，确认服务健康（无需外部 key）
- [ ] 重排模型 `bge-reranker-v2-m3`：内置容器，确认服务健康

## D. 导入文档（← 接入你们的内部文档）
- [ ] 创建第一个知识库
- [ ] 选导入方式：URL / Sitemap / RSS / 离线文件（md/word/pdf）
- [ ] 导入后测试：AI 问答能引用到刚导入的内容

## E. 运维
- [ ] 记录安装器打印的**全部端口/URL**
- [ ] 确认 Docker 卷备份策略（数据库/对象存储等）
- [ ] 需要对外访问的话，前面加反代 + HTTPS，并按内网安全策略限制访问

## F. 合规确认
- [ ] 对话模型确实指向内网网关，LLM 流量不出网
- [ ] AGPL-3.0：若有源码改造 + 对外服务，走一下法务；纯内部自用无碍
